# Projetos da Disciplina de Ciência de Dados

Este repositório armazena os notebooks e trabalhos desenvolvidos para a disciplina de Ciência de Dados (Grupo 5). O foco é a aplicação prática de algoritmos de machine learning, pré-processamento e avaliação de modelos.

## Conteúdo

Os notebooks deste repositório cobrem diversas técnicas fundamentais da Ciência de Dados:

* **`Grupo5_RegressaoLinearLogit.ipynb`**: Análise e aplicação de modelos de Regressão Linear e Regressão Logística (Logit).
* **`Grupo5_Notebook_ArvoreDecisao_KNN_Regressao.ipynb`**: Implementação de algoritmos de aprendizado supervisionado, incluindo Árvore de Decisão, K-Nearest Neighbors (KNN) e outros modelos de Regressão.
* **`Clustering_Notebook_GRUPO5.ipynb`**: Estudo e aplicação de técnicas de aprendizado não supervisionado para Clusterização de dados.
* **`Grupo_5_RegrasAssociação.ipynb`**: Análise de padrões de dados e implementação de Regras de Associação.
* **`Grupo5_AvaliacaoModelos_FeatureEngineering_KNN_CV.ipynb`**: Notebook focado em Engenharia de Features (Feature Engineering), métodos de Validação Cruzada (Cross-Validation) e avaliação de performance de modelos.
* **`PreProcessamento_KNN_Stroke_Grupo5.json`**: Arquivo JSON de suporte, possivelmente contendo dados ou configurações de pré-processamento para o dataset "Stroke".
